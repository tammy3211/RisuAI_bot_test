# Lorebook

로어북은 캐릭터의 세계관, 설정, 관계 등을 정의하는 데이터베이스입니다.

## 항목 예시

### 세계관
- **키워드**: 세계관, world, setting
- **내용**: 이 이야기는 미래의 사이버펑크 세계를 배경으로 합니다. 거대 기업들이 도시를 지배하고, 네온 불빛이 밤하늘을 밝힙니다.

### 캐릭터 배경
- **키워드**: 과거, background, history  
- **내용**: 주인공은 전직 해커로, 과거의 실수로 인해 현재의 삶을 살고 있습니다.

### 주요 장소
- **키워드**: 나이트 시티, Night City
- **내용**: 네온 불빛과 홀로그램 광고로 가득한 거대 도시. 상층부는 화려하지만 하층부는 빈곤과 범죄로 얼룩져 있습니다.

## 로어북 사용 팁

1. **키워드를 명확하게** - 대화에서 자연스럽게 언급될 만한 키워드를 설정하세요
2. **내용은 간결하게** - 너무 긴 내용은 컨텍스트를 낭비합니다
3. **우선순위 설정** - 중요한 항목은 항상 활성화(Always Active)로 설정하세요
4. **정규표현식 활용** - useRegex 옵션으로 더 정교한 매칭이 가능합니다

## CBS 활용 예제

로어북 내용에도 CBS를 사용할 수 있습니다:

```
{{char}}는 {{gettempvar::location}}에 살고 있으며, 
{{#if {{equal::{{time}}::night}}}}
밤에는 주로 클럽에서 시간을 보냅니다.
{{/if}}
```

## Regex 패턴 예제

- 특정 인물 이름: `\b(Alice|Bob|Charlie)\b`
- 장소 언급: `\b(카페|레스토랑|공원)\b`
- 감정 표현: `\b(행복|슬픔|화남|기쁨)\b`
