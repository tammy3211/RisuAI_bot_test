# Regex Scripts

This folder contains regex replacement scripts for text processing.

## Available Scripts

- **action_emphasis**: Emphasizes action text with special formatting
- **double_space_fix**: Removes duplicate spaces
- **emoticon_to_text**: Converts emoticons to text descriptions
- **profanity_filter**: Filters inappropriate language
- **time_normalization**: Normalizes time expressions

All script outputs are stored in the `out/` directory.
